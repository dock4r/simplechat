### Prerequisites

* Java 1.8
* NodeJS 11+
* Docker
* Cassandra 3
* RabbitMQ 3
* Redis 5

## Development

### Use homebrew (macOS only)

`$ brew cask install docker`

(Don't know homebrew? [brew.sh](https://brew.sh/))
